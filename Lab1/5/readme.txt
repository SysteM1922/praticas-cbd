Ficheiro Chat.java

Set users -> Set que guarda todos os utilizadores que fazem log in no chat

Set ordenado user+"_contacts" -> Set que guarda os contactos registados pelo utilizador guardado na variável "user"

List dest+"_msgs" e List user+"_msgs" -> Listas que guardam as mensagens enviadas de um utilizador "user" para um destinatário "dest", as mensagens contêm o emissor, a hora a que esta foi enviada e o conteúdo da mensagem
